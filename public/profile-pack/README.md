# 새세공 330 프로필 이미지

최종 프로필 이미지 세트입니다. 모든 이미지는 같은 마크를 사용하며, 배경색과 대비만 다릅니다.

## 파일 구성

- `svg/`: 원본 벡터 파일
- `png/400/`: X/Twitter 프로필 업로드용 PNG
- `png/512/`: 범용 프로필/아이콘용 PNG
- `png/1024/`: 고해상도 보관/재가공용 PNG

## 이미지 목록

- `01_default_dark`: 기본 다크 버전
- `02_default_light`: 기본 라이트 버전
- `03_small_dark`: 작은 표시 크기용 다크 버전
- `04_small_light`: 작은 표시 크기용 라이트 버전
- `05_high_contrast_dark`: 고대비 다크 버전
- `06_high_contrast_light`: 고대비 라이트 버전
- `07_soft_graphite`: 부드러운 그래파이트 버전
- `08_soft_warm`: 부드러운 웜 라이트 버전
- `09_midnight_navy`: 미드나이트 네이비 버전
- `10_transparent_dark_mark`: 투명 배경의 다크 마크
- `11_transparent_light_mark`: 투명 배경의 라이트 마크

## 해상도 기준

- `400x400`: X/Twitter 공식 권장 프로필 이미지 크기입니다.
- `512x512`: 여러 서비스와 아이콘 워크플로에서 쓰기 좋은 범용 정사각 해상도입니다.
- `1024x1024`: 업로드 필수 크기는 아니며, 고해상도 마스터로 보관하기 위한 크기입니다.

X/Twitter에 바로 올릴 때는 `png/400/01_default_dark.png` 또는 `png/400/02_default_light.png`를 우선 사용하면 됩니다.
