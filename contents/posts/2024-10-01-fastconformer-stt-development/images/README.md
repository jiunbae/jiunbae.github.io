# FastConformer STT Architecture

This directory contains images for the FastConformer STT development blog post.

## Files
- `architecture.png`: Main architecture diagram showing the FastConformer block structure and data flow
- `performance_chart.png`: Performance comparison chart showing WER/CER improvements
- `training_pipeline.png`: Training pipeline and data preprocessing flow

## Image Generation
Images are generated using Mermaid diagrams and converted to PNG format for better blog integration.

## Usage
Images are referenced in the blog post using relative paths:
```markdown
![Architecture](./images/architecture.png)
```
